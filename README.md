🚀 DecodeLabs Project 1 – Data Cleaning Analytics Dashboard

📊 Project Highlights

✅ Cleaned and standardized sales data using ISO 8601 date formatting
✅ Removed duplicates and validated data integrity
✅ Created interactive filters for City and Order Year
✅ Built KPI cards to track revenue, orders, and compliance metrics
✅ Designed insightful visualizations for revenue trends and category performance

🔍 Key Insights

* Total Revenue: $585.42K
* Unique Orders: 1,199
* ISO 8601 Compliance: 100%
* Top Performing Hub: Bengaluru
* Technology category contributed the highest sales share
